# task 4 - data sources
* autopsy supports the following disk image formats:
    * raw single : *.img, *.dd, *.raw, *.bin
    * raw split : *.001, *.002, *.aa, *.ab, ...
    * encase : *.e01, *.e02, ...
    * VMs : *.vmdk, *.vhd
