# Task 2 - Types of Cryptography
* Symmetric
  * encryption
  * decryption
  * key
  * encryption and decryption keys are the same
  * fast
* Asymmetric
  * public key used for encryption
  * private key used for decryption
  * encryption and decryption keys are different
  * slower than symmetric
